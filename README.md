# Vite Focale

A minimal gallery website using **React**, **Tailwind CSS** working with **Vite**.

## Development

```bash
git clone https://github.com/area44/vite-focale
cd vite-focale
pnpm install
pnpm dev
```
Open your web browser and navigate to http://localhost:5173.
